Windows下使用WSL(Linux子系统) + Visual Studio Code配置C/C++配置环境的教程

由于个人原因更换了新电脑，然后需要在新电脑上配置开发环境，于是使用了Visual Studio Code + WSL来配置简便的Linux C/C++开发环境

安装Visual Studio Code

这个过程非常简单，按照网上的教程一步步的进行即可。

Visual Studio Code安装必要插件

强烈推荐Microsoft官方开发的C/C++插件。

安装WSL(Linux子系统)

在Microsoft官方商店中搜索Linux，选择喜欢的系统类型进行下载即可。笔者选择的是Ubuntu，Microsoft商店中下载的Ubuntu默认是18.04版本。下载完成后可以通过开始菜单打开，跟着提示一路完成即可。

WSL安装必要的编译环境

WSL默认只带了gcc编译器，因此想要配置C++编译环境的同学需要自行下载g++编译器以及gdb调试工具。关于这个可以查阅Microsoft官方文档，不再过多赘述。
